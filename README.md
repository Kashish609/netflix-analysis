# netflix-analysis
This project analyzes the Netflix dataset to explore trends, patterns, and insights about movies and TV shows available on the platform. It uses Python, Pandas, Matplotlib, and Seaborn for data visualization and analysis, along with TF-IDF Vectorization for text-based insights.


This project explores and visualizes the Netflix Movies and TV Shows dataset from Kaggle, aiming to uncover key trends and patterns in the content available on the platform.
